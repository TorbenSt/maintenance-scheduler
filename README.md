## About the project

This repository contains a **beta-stage SaaS application** built with Laravel 12.

The focus of this project is not a finished UI, but a **clean, test-driven domain model**
for recurring maintenance contracts and automated appointment scheduling.

### Implemented features
- Multi-tenant company structure
- Recurring maintenance contracts (interval-based)
- Automatic task generation per interval
- Appointment proposals with secure signed links
- Public accept/reject flow without authentication
- Queued email delivery
- Extensive Pest test coverage

### Planned features
- Customer UI (Livewire / Volt)
- Technician capacity & route optimization
- Billing (Stripe / Paddle)
- Reminder & follow-up automation
